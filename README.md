# Estudo de Caso 

## Descrição do Projeto
Este projeto é uma aplicação full-stack, composta por uma versão web e mobile, que consome uma API REST de produtos. A aplicação permite listar produtos e visualizar seus detalhes.

A versão web foi desenvolvida com React e Vite, e a versão mobile com React Native e Expo.

## Estrutura do Projeto
meu-estudo-caso/
├── web/ # Projeto React com Vite
├── mobile/ # Projeto React Native com Expo
└── README.md # Este arquivo

## Tecnologias Utilizadas

### Web
- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [React Router DOM](https://reactrouter.com/en/main) - para roteamento
- [Axios](https://axios-http.com/) - para consumo da API
- [React Bootstrap](https://react-bootstrap.github.io/) - para estilização

### Mobile
- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/) - para navegação
- [Axios](https://axios-http.com/) - para consumo da API
- [React Native Paper](https://callstack.github.io/react-native-paper/) (opcional) - para estilização

## Funcionalidades

### Web
- Página inicial com listagem de produtos
- Página de detalhes do produto
- Navegação entre rotas

### Mobile
- Tela inicial com listagem de produtos
- Tela de detalhes do produto
- Navegação entre telas
